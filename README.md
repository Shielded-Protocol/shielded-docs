## What this does

This repository contains the source code for the Shielded Protocol documentation site, built with Astro and Starlight. It covers everything from technical architecture and ZK circuit design to integration guides for developers.

View the live documentation at [docs.shielded-protocol.org](https://docs.shielded-protocol.org).

## Quickstart

```bash
npm install
npm run dev
```
